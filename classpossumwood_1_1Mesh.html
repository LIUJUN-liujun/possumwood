Detailed Description

A COW mesh data structure - calling edit() allows for mesh editing, but also makes sure that any underlying data are deduplicated. Otherwise, the copying of Mesh instances produces read-only shallow copies. Currently, there is no protection for mis-use of the COW - storing a reference to the MeshData datastructure is a terrible idea.
